Dance Library

A web app designed for dancers of all levels to create their own virtual library of dance classes from their favorite instructors. Users can browse, search, filter classes and instructors to add to their personal library. Users can edit and delete classes and instructors they added and favorite classes to save them to their profile for later viewing.

Project Link
Github Repository

I created Dance Library, a fullstack React application, independently as a tool for dancers to be able to create a personal library or "studio" to take classes from their favorite instructors and explore new styles. A basic CRUD application, users can learn about offerings by searching, filtering, and adding classes or instructors. Users can also favorite classes and edit or delete classes or instructors that they added to the library.

Technologies:

React, Redux, PostgreSQL, Express, Node, JavaScript, CSS, Material UI, Bcrypt, Cypress
