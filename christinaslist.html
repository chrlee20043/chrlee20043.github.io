Christina's List

A web app where users can sell and purchase goods. Users can ceate posts to sell unwanted items, edit and delete their posts. Users can message sellers to inquire about items for sale and view their posts and sent messages on their profile page.

Project Link
Github Repository

Christina's List is a Craigslist clone and front-end application using a RESTful API created through Fullstack Academy. Users can interact with the application with basic CRUD operations to sell, purchase, and message users.

Technologies:

React, Redux, JavaScript, CSS, RESTful API
